Site Purpose

The purpose of this website is to provide current updates on area weather forecasts for the public. In doing this it will need to display current weather conditions in an easy to understand way. It will convey accurate weather conditions for various locations, display maps, and show images related to the current conditions. It will also display forecasts for upcoming weather conditions that are accurate according to the current meteorological standards. The site needs to have an easy-to-use user interface that is intuitive to first-time website visitors. The weather conditions displayed must update in a timely manner and correspond to real-time conditions. Users of the site should feel comfortable trusting that the information conveyed is accurate.
